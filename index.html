DRAFT - ANS-REG-001: Agent Name Registration Protocol Status: Research Phase Date: January 2025 Authors: ANS Registry Research Team

Abstract

This document proposes a registration protocol for Agent Name System (ANS) identifiers using the agent:// URI scheme. The protocol defines registration, renewal, transfer, and resolution mechanisms for AI agent naming.

Example Agent Name Structure

agent://support.company.com agent://api.payments.stripe.com agent://translate.docs.google.ai

Proposed Registration Flow

1. Domain Verification (/.well-known/agent.json) 2. Cryptographic Proof of Control 3. Registration in Distributed Ledger 4. Global Propagation to Resolvers
